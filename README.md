## â“ Troubleshooting

**Game won't start?**
- Make sure Windows Defender isn't blocking it
- Right-click the .exe and select "Run as Administrator"

**No sound?**
- Check your volume settings
- Make sure Windows hasn't muted the app

**Lost my saves?**
- Check: `C:\Users\[YourName]\AppData\Roaming\911-dispatcher\saves\`
- Save files end with `.savegame`

**Want to uninstall?**
- Use Windows Settings > Apps > 911 Dispatcher > Uninstall
- Or find it in Control Panel > Programs and Features
